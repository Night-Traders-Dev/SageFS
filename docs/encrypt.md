# Encryption Layer

## Overview
The Encryption Layer provides per-file encryption with hardware acceleration.

## Key Features
- **File Data Encryption**: AES-256-XTS.
- **Filename Encryption**: AES-256-CTS.
- **Key Derivation**: Argon2 / PBKDF2 from user passphrases.
- **Hardware Acceleration**: AES-NI integration for near-zero overhead.
